# 完美实时响应的Loading

#### 介绍
有时候，一些网站因为功能比较强大，导致需要加载的文件特别多，让人总会有卡顿的感觉。
这时候，怎么实现快速的显示加载页面呢？
简单，当点击url后，就在当前页面立刻显示loading动画。同时，跳转页面上，也需要最优先显示对应的loading动画。
这样，当前后结合起来时，效果就达成了。不会感到卡顿。因为响应都是实时的。所以叫做完美实时响应的Loading。
当然，代码是基于layui的。

Sometimes, some websites, due to their powerful functions, require loading a large number of files, which often gives people a feeling of sluggishness.
At this point, how can we achieve a quick display of the loading page?
Simply, when the URL is clicked, a loading animation should be displayed immediately on the current page. At the same time, on the redirected page, the corresponding loading animation should also be displayed with the highest priority.
When the front and back are combined, the effect is achieved. There is no stuttering sensation. Because the response is real-time. So it is called Loading with perfect real-time response.
Of course, the code is based on layui.

git pull或下载后查看loading.html页面即可看到测试、演示和用法。
